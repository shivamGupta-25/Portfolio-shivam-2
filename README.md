# Shivam Raj Gupta — Portfolio

Personal portfolio website built with **Next.js**, **Tailwind CSS**, **Shadcn UI**, and **Magic UI**. Deployed on Vercel.

🔗 **Live:** [shivamgupta-25.vercel.app/](https://shivamgupta-25.vercel.app/)

## Stack

- [Next.js](https://nextjs.org/) — Framework
- [Shadcn UI](https://ui.shadcn.com/) — Components
- [Magic UI](https://magicui.design/) — Animations
- [Tailwind CSS](https://tailwindcss.com/) — Styling

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev
```

All content is managed from a single config file: [`src/data/resume.tsx`](./src/data/resume.tsx)
